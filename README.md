# Simple Node.js Web Server

This project is a basic web server built using Node.js core modules. It demonstrates how to create an HTTP server, serve an HTML file, handle multiple routes, and return a 404 response for invalid routes.

## Features
- Serves an HTML home page
- Displays a text response for the `/about` route
- Handles unknown routes with a 404 error
- Uses Node.js core modules only (`http`, `fs`)
- Uses ES module syntax

## How to Run the Server
1. Make sure Node.js is installed
2. Navigate to the project folder
3. Run the server:
4. Open a browser and visit:
- `http://localhost:3000/` for the home page
- `http://localhost:3000/about` for the about page

## Expected Behavior
- The home page loads an HTML file
- The about page displays a simple text message
- Invalid routes return a 404 "Not found" response
